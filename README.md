# Description

This project is to solve the simple and complex assignment 

## Installation
* Clone the project
* Run test
  * Run all assignemnts
```bash
mvn test
```

   * Run simple assignment
```bash
mvn test -Dtest=AssignmentSimpleRunner 
```
   * Run complex assignment
```bash
mvn test -Dtest=AssignmentComplexRunner
